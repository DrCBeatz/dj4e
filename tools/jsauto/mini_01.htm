Run a local copy of the mini_django web server on your computer

Pre-requisites (feel free to use AI to help with these pre-requisites):

  • Install python on your computer so that it works at the command line
  • (Optional) Install git on your computer

For this assignment you need to go to 

https://github.com/csev/mini_django
and download the source code into a folder as a ZIP and unzip it or check out the source code into a folder using git.

Follow the instructions `README.md` to get the server running on your computer at port 9000 with he autograder enabled: 

cd mini_django
python runserver.py 9000 autograder
Test your application by navigating to http://localhost:9000.

Once the application is running, use this autograder to exercise the application and earn a grade.

Important: The autograder does not work on the Safari Browser.
